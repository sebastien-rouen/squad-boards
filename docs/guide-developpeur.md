# Guide Developpeur

> Visualisez votre sprint, trouvez vos tickets et suivez votre charge.

---

## Votre utilisation quotidienne

### Sprint Board (`2`)
- Selectionnez **votre equipe** dans la sidebar (carre colore)
- Voyez vos tickets dans les colonnes : A faire → En cours → Revue → Done
- Mode **swimlanes** : vos tickets regroupes par assignee
- Mode **liste** : vue compacte triable pour une vision rapide

### Filtres rapides
- **Non assignes** : tickets a prendre
- **Bloques** : impediments a remonter au Scrum Master
- **Critique/High** : priorites du jour

### Detail ticket
- Cliquez un ticket → modal avec tout le detail
- **Description riche** : gras, italique, liens, code, listes (ADF parse)
- **Liens** : tickets lies (bloque, relate a, etc.)
- **Commentaires** : ajoutez directement depuis la modal
- **Naviguer** : fleches ← → pour passer au ticket suivant

## Creer un ticket

1. Touche `N` ou bouton **+ Nouveau**
2. Remplissez : titre, type, equipe, leader, contributors, points
3. Ajoutez un epic parent si pertinent
